Supplementary-01-001_자기앞수표발행_Definition

목차​

  • 1.요건
  • 2.업무흐름
  • 3.테스트케이스

1. 요건

1.1 개요​

  • 자기앞수표를 발행
    • 자기앞수표 : 은행이 지급인으로 정하여 발행한 수표

1.2 항목내용​

1.2.1 수표정보​

  • 항목내용
    • 수표권종
      • 정액권 : 10만원권, 30만원권(단종), 50만원권, 100만원권
      • 일반권 : 500만원권, 1000만원권, 5000만원권, 비정액권
    • 수표번호
    • 매수
    • 발행금액
  • 제한내용
    • 중요증서정보와 체크
      • 미등록 수표는 발행할 수 없음
    • 발행금액은 별단계정에 수납

1.2.2 의뢰인정보​

  • 항목내용
    • CTR보고자료를 위한 정보
  • 제한내용
    • 창구,ATM과 같이 실물이 발생하는 매체거래가 대상

1.2.3 대리인정보​

  • 항목내용
    • 의뢰인에게 권한을 위임받아 대신 거래를 할 수 있도록 허용
  • 제한내용
    • 창구거래만 대상

1.2.4 현금대체구분​

  • 항목내용 : 현금, 대체

1.3 기능제한내용​

1.3.1 자기앞수표발행수수료​

  • 면제요건 정리해야 함

1.4 출력물​

1.4.1 단말​

  • 자기앞수표 : 비정액권인 경우 금액출력
  • 전표
  • 영수증

1.5 회계처리​

1.5.1 원점거래​

  • 관리점
    • 차변 : N/A
    • 대변 : 거래금액, 수수료

1.5.2 NET점거래​

  • N/A
